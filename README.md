# ARQJS

JS Wrapper For the A.R.Q. API.

## Installation

```sh
$ npm install arq-js
```

## Usage example

Get direct Deezer song link:

```js
const { ARQ } = require("arq-js");

const arq = new ARQ("http://yourArqApi.com");

arq.deezer("Ciao Bella", 1).then(console.log);
```
