# ARQJS

JS Wrapper For the A.R.Q. API.

## Installation

```sh
$ npm install arq
```

## Usage example

Get direct Deezer song link:

```js
const { ARQ } = require("arq");

const arq = new ARQ("http://yourArqApi.com");

arq.deezer("Ciao Bella").then(console.log);
```
